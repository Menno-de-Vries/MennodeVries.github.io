Blog 1: De bouw van mijn portfolio website

In deze blog leg ik uit wie ik ben, waarom ik deze website heb gebouwd en welke keuzes ik heb gemaakt tijdens het bouwen.

Wie ik ben

Ik ben Menno de Vries. Met een achtergrond in MBO4 Game Development heb ik veel ervaring opgedaan met Unity en C#. Momenteel studeer ik verder in Cybersecurity & Cloud aan de Hogeschool Utrecht.

Gedachte achter deze website

Deze website is gemaakt als centrale plek voor mijn projecten, ontwikkeling en leerproces. Ik wilde een overzichtelijke en professionele basis neerzetten die ik kan blijven uitbreiden met nieuwe projecten en blogposts.

Belangrijk voor mij was dat de website duidelijk laat zien waar mijn kracht ligt: Unity/C# en mijn groei richting infrastructuur, automatisering en netwerkbeveiliging.

Wat ik tijdens de bouw heb gedaan

Structuur en layout

Alle pagina's zijn gelijkgetrokken met een consistente header, navigatie, kleuren en kaartlayout voor een professionele uitstraling.

Project presentatie

Voor projecten heb ik aparte secties gemaakt, inclusief interactieve elementen en een templatepagina om screenshots, video en code te tonen.

Contact en bereikbaarheid

Het contactformulier is werkend gemaakt en de website is gekoppeld aan een eigen domein voor een nette presentatie.

Doorontwikkeling

Deze website blijft groeien. Ik voeg stap voor stap projecten, technische uitleg en concrete leerresultaten toe.
